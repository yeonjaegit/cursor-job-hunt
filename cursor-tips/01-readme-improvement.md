## 💡 Cursor 활용 방법

### 1. 프로젝트 코드베이스 분석 요청

**프롬프트:**
```
이 프로젝트의 모든 코드를 분석해서 다음 정보를 정리해줘:
1. 핵심 기술과 구현 방식
2. ERD 설계 특징
3. 최적화나 성능 개선 부분
4. 보안/인증 관련 구현
5. 면접에서 물어볼만한 기술적 깊이
```

**결과:**
- Cursor가 전체 코드베이스를 탐색하여 핵심 포인트 추출
- N+1 쿼리 최적화, MQTT 아키텍처 등 기술적 도전 과제 발견

---

### 2. 아키텍처 다이어그램 생성

```
이 프로젝트의 시스템 아키텍처를 Mermaid 다이어그램으로 그려줘
```

**결과:**

### 3. 코드 예시 자동 추출

**프롬프트:**
```
이 프로젝트에서 가장 핵심적인 코드 3개를 골라서 README에 넣을 수 있게 정리해줘
```

**결과:**
- Write-Heavy 피드 생성 로직
